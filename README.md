## How to run the project
1. clone the project
```angular2html
git clone https://github.com/ezedinff/sample-project
```
2. got to sample-project directory
  ```angular2html
cd sample-project
```
3. run **npm install**
4. after successfully installed the dependencies run ***npm start***
